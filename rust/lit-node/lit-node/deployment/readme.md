# Deployment Folder Purpose 

These files are mostly used for setting up a centralized dev server for deploying the lit-node.

More docs are here: https://www.notion.so/litprotocol/Setting-up-a-centralized-dev-network-17a6f449b041807288bcfda4c7d6e5c7?pvs=4


## Additional files

- "docker_experiments" - contains the docker files for the node
- "sgx" - contains the files for the node inside of Intel SGX
- "libs" - contains secrets for Intel SGX
