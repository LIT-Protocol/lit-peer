# lit-vrf
A curve agnostic VRF implementation following [RFC9381](https://datatracker.ietf.org/doc/rfc9381/) and
[Fully Distributed VRF](https://eprint.iacr.org/2020/096.pdf).

Also, this extends the VRF scheme to support the [eVRF](https://eprint.iacr.org/2024/397.pdf)
and the [Distributed VRF](https://eprint.iacr.org/2024/1130.pdf).
